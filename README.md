# chatbox
<h1>ChatBox created using MERN stack, Jsonwebtoken, cookie-parser and socket.io </h1>

![image](https://github.com/kulender-Annapureddy/chatbox/assets/153791259/0a4b1904-abc2-4494-a99d-e3f46f6a5e7e)

<h3>Setup .envfile</h3>
<pre>
  PORT=...
  MONGO_URL='......'
  JWT_SECRET=....
  NODE_ENV=.......
</pre>

<h2>Build the app</h2>
<p>npm run build</p>

<h2>Start the app</h2>
<p>npm start</p>
